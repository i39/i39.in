# Сайт ООО «Инновация»

Это статический сайт на Hugo. Основной контент редактируется без изменения шаблонов:

- `content/_index.md` — главный экран, вводный текст, заголовок формы и список слайдов `hero_slides`;
- `content/about.md` — страница и блок «О компании»;
- `content/services.md` — страница и блок услуг;
- `content/contact.md` — контакты и ссылка на карту;
- `hugo.toml` — название сайта, меню и путь к фоновому изображению;
- `static/images/hero.png` — фоновое изображение главного экрана.

После изменений запустите локальную проверку:

```bash
cd i39
hugo server -D
```

Собрать сайт для публикации:

```bash
hugo --minify
```

Готовые файлы появятся в `public/`. Форма подготовлена для Netlify Forms (`data-netlify="true"`). На другом хостинге подключите обработчик формы или сервис отправки заявок.
